## üíª Getting Started

### Prerequisites

Ensure you have Node.js and npm (or yarn/pnpm) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd my-react-ts-app
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Running Locally

Start the development server:

```bash
npm run dev
```

Visit `http://localhost:5173` (or the URL shown in the terminal) to view the app.
